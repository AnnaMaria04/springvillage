# Изображения сайта

Замените эти файлы реальными фотографиями. До замены на их месте
показывается нейтральный фон-заглушка (сайт не ломается).

Рекомендации:
- Формат: `.jpg` (фото) — имена должны совпадать точь-в-точь.
- Hero: горизонтальное, минимум 2000×1200, вес < 400 КБ (сожмите через squoosh.app).
- Карточки: соотношение примерно 4:3 или 3:2, минимум 1000 px по ширине.
- Только реальные фото коттеджа/озера/бани — без стоков и AI-картинок.

## Нужные файлы

### Главная (homepage)
- `hero.jpg` — большое фото коттеджа/озера на весь экран (главный экран)
- `stay.jpg` — коттедж для блока «Размещение» и карточки бронирования
- `booking.jpg` — фон для финального блока «Приезжайте к озеру»
- `exp-sauna.jpg` — баня (превью впечатлений)
- `exp-lake.jpg` — озеро / байдарки (превью впечатлений)
- `exp-forest.jpg` — лес (превью впечатлений)

### Галерея (Gallery)
- `gallery-exterior.jpg` — коттедж снаружи (большое)
- `gallery-interior.jpg` — интерьер / гостиная
- `gallery-sauna.jpg` — баня
- `gallery-lake.jpg` — вид на озеро
- `gallery-sunset.jpg` — закат с террасы
- `gallery-winter.jpg` — зимний кадр

### Активности — Лето
- `activity-kayak.jpg`, `activity-fishing.jpg`, `activity-swim.jpg`,
  `activity-bike.jpg`, `activity-bbq.jpg`, `activity-games.jpg`

### Активности — Зима
- `activity-banya.jpg`, `activity-ski.jpg`, `activity-skate.jpg`,
  `activity-icefish.jpg`, `activity-forest.jpg`, `activity-fireplace.jpg`
